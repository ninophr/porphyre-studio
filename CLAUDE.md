# Porphyre Studio

Site d'agence porphyre.studio — pages marketing, tunnel de commande, dashboard admin, backend automatisé (Stripe, emails, briefs).
Stack : HTML + Tailwind CDN + Vercel Serverless Functions (JS).

## Contexte projet

Le vault Obsidian ~/Documents/Obsidian Vault/ contient :

- L'état du projet et ce qui reste à faire → Agence/Porphyre Studio.md
- Les objectifs → Objectifs/
- Les clients → Projets/

Consulte ces notes quand tu as besoin de contexte.

## Règles

- Travailler en local, commit + push quand un bloc est terminé
- Ne jamais déployer manuellement sans demander
- Après chaque bloc de travail terminé, mettre à jour automatiquement le vault Obsidian :
  - Cocher les tâches faites dans Agence/Porphyre Studio.md
  - Mettre à jour les status (à vérifier → fait, manquant → fait, etc.)
  - Cocher les critères de succès dans Objectifs/Avril 2026.md si atteints
  - Ajouter une ligne dans l'historique du client si un projet client a été touché
  - Ne jamais demander avant de mettre à jour — le faire automatiquement
